# スクショット

```
  ____            _               _   _           
 / ___|  ___ _ __(_)_ __   __ _  | |_(_) ___  _ __ 
 \___ \ / __| '__| | '_ \ / _` | | __| |/ _ \| '__|
  ___) | (__| |  | | | | | (_| | | |_| | (_) | |   
 |____/ \___|_|  |_|_| |_|\__,_|  \__|_|\___/|_|   
```

スクショットは、画面の指定範囲を撮影し、そのままプレビュー表示するだけのシンプルなツールです。
保存は任意で、必要なときだけ行えます。

## まず何ができる？

| これがしたい | スクショットでできること |
| --- | --- |
| 画面の一部だけ撮りたい | クリック＆ドラッグで範囲指定 |
| 拡張画面もまとめて撮りたい | マルチモニター環境に対応 |
| すぐ見たい | 撮影後に自動でプレビュー表示 |
| 保存は必要なときだけ | 右クリックから保存を選ぶだけ |
| ちょっと書き込みたい | `Ctrl + ドラッグ`で赤線を描画 |

## 主な機能

- クリック＆ドラッグで範囲指定してスクリーンショット
- マルチモニター環境で全ディスプレイからキャプチャ可能
- 撮影後は別ウィンドウでプレビュー表示（自動保存はしません）
- 右クリックメニューで「保存 / ズームイン / ズームアウト」
- `Ctrl + ドラッグ`で赤い線を描画
- 追加撮影ボタンで連続撮影

## 操作早見表

| 操作 | 動作 |
| --- | --- |
| 左クリック＆ドラッグ | 範囲を選択して撮影 |
| 右クリック | メニュー（保存 / ズームイン / ズームアウト） |
| `Ctrl + ドラッグ` | 赤い線で描画 |
| 追加撮影ボタン | 新しいスクショを撮影 |
| `Esc` / 右クリック（撮影画面） | キャンセル |

## 動作環境

- Windows 10 以降
- Python 3.9+

## 使い方

1. 依存関係をインストールします。

```
pip install -r requirements.txt
```

2. 起動します。

```
python main.py
```

3. 画面でドラッグして範囲を指定します。

4. プレビュー画面で確認・編集します。

## できること / できないこと

| できること | できないこと |
| --- | --- |
| 範囲指定スクショ | 自動保存 |
| 撮影直後のプレビュー | 自動連番保存 |
| 必要なときだけ保存 | 編集の取り消し（Undo） |
| 赤線の書き込み | 文字入力や図形描画 |

## 保存について

撮影後は自動保存されません。必要な場合だけ右クリックメニューの「画像の保存」を使用してください。
